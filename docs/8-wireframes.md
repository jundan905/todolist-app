**문서 버전:** 1.0.0

**기준 문서:** 도메인 정의 (v0.2.0), PRD (v1.0.3), 사용자 시나리오 (v1.0.0), 프로젝트 구조 (v1.0.2)

## 1. 와이어프레임 개요

### 1.1 애플리케이션 구조

TodoList 애플리케이션은 두 개의 주요 영역으로 구성된 **7 개 핵심 화면**으로 이루어져 있습니다:

```
┌─────────────────────────────────────────────────────────────┐
│                    TodoList 앱 구조                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────┐    ┌────────────────────────────┐  │
│  │   인증 플로우       │    │   메인 앱 (보호됨)         │  │
│  │   (비인증)          │    │   (인증됨)                 │  │
│  │                     │    │                            │  │
│  │  • 로그인 페이지    │───▶│  • 투두 리스트 페이지      │  │
│  │  • 회원가입 페이지  │    │  • 투두 상세 모달          │  │
│  │                     │    │  • 투두 생성/수정 모달     │  │
│  └─────────────────────┘    │  • 사용자 메뉴             │  │
│                             └────────────────────────────┘  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 화면 인벤토리

| 화면 ID | 화면 이름 | 타입 | 인증 필요 | UC 참조 |
|-----------|-------------|------|---------------|--------------|
| SCR-01 | 로그인 페이지 | 전체 페이지 | 아니오 | UC-02 |
| SCR-02 | 회원가입 페이지 | 전체 페이지 | 아니오 | UC-01 |
| SCR-03 | 투두 리스트 페이지 | 전체 페이지 | 예 | UC-05 |
| SCR-04 | 투두 생성 모달 | 모달/오버레이 | 예 | UC-04 |
| SCR-05 | 투두 상세 모달 | 모달/오버레이 | 예 | UC-06, UC-07 |
| SCR-06 | 투두 수정 폼 | 모달/오버레이 | 예 | UC-07 |
| SCR-07 | 사용자 메뉴 | 드롭다운 | 예 | UC-03 |
